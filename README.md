# musicRecommend
## 项目说明
+ 本项目为OOP大作业，目的是实现一个基于文本匹配的音乐推荐系统
+ 请大家把代码放上来
+ 可以把项目fork到自己的github下，使用github网页版不需要会git命令，可以参看github自己的教程。

+ 所有类请进行完整的封装。提供完整的set和get接口。
+ 周三之前请上传第一版

+ 使用VS的nuget工具
+ 建议选择140vc版本。安装时有三个包
   -  leveldb
   -  crc32c
   -  snappy
